

# Gatsby

This are sample Gatsby websites

## Install and Run

Make sure that you have the Gatsby CLI program installed:

```sh
npm install -g gatsby-cli
```

Move to folder 
```sh
cd demo-1
```
or 

```sh
cd demo-2
```

Install the dependencies:

```sh
npm install
```

Develop your site locally:
```sh
gatsby develop
```
